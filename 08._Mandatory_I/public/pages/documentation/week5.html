Week 5 - Export / Import

Import and export

Import and export are two ways to include a module in another module. The import statement can be used to import functions, objects, or primitive values that have been exported from an external module. The export statement is used when creating JavaScript modules to export functions, objects, or primitive values from the module so they can be used by other files with the import statement.

In HTML you can use the <link> tag to import a CSS file and the <script> tag to import a JavaScript file. The script tag can also be used to import a module. <script type="module" src="module.js"></script>
This tells the browser that the linked JavaScript file (module.js in this case), should be treated as a module. Using modules helps organize and modularize your JavaScript code, making it easier to manage and reuse across different parts of your application.

In Node.js you can use the require() function to import a module, which is the CommonJS way of importing modules. When importing express for example, we could write:
const express = require('express');
The require() function returns the module.exports object, which contains all the exports from the module that is being required.
