What is node.js?

A cross-platform, to have javascript both for frontend and backend, so that developers only need to know one language. Used for many different kinds of applications.

npm

Node Package Manager, a package manager for JavaScript programming language. It is the default package manager for the JavaScript runtime environment Node.js.
npm -i express - installs the express package.

Express

Express, which we use this semester, is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.

Datatypes in JavaScript

JavaScript has 7 data types:

  • Number
  • String
  • Boolean
  • Object
  • Undefined
  • Null
  • Symbol

typeof operator can be used to find the data type of a JavaScript variable.

Example:
console.log(typeof "hello"); - This will output "string".

Type Coercion

JavaScript is a loosely typed language, which means that it can convert a variable from one type to another automatically.

Example:
console.log(1 + "2"); - This will output "12" because the number 1 is coerced into a string.

To avoid type coersion, use:
=== instead of ==
!== instead of !=
when comparing values and types.

Also use explicit type conversion when needed.
Convert values explicitly using functions like: parseInt(), parseFloat(), Number(), String(), or Boolean(). It also makes the code more readable.

Declare variables with their intended data types explicitly. This helps prevent unexpected type coercion.

Strings
Different ways of definings strings:

String methods:
toUpperCase() - converts a string to uppercase letters.
toLowerCase() - converts a string to lowercase letters.
trim() - removes whitespace from both ends of a string.
split() - splits a string into an array of substrings.

Designing a REST API

A properly named resource makes an API simple to use and intuitive.

The 3 REST conventions:
1. Using the right HTTP verbs
2. Ordering your URL correctly
3. Naming (noun, plural) that maps to collections in your system
